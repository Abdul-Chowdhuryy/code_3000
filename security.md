# Security Overview

## Intended Users
The intended users of this repository are the instructor and students in the course. The repository contains coursework and programming assignments used for educational purposes.

## Security Risk Assessment
If the code or data in this repository fell into the wrong hands, the main concern would be unauthorized access to assignment solutions or coursework materials. This could lead to academic dishonesty or plagiarism. However, the repository does not contain sensitive personal or financial data, so the overall security risk is low.

## Steps Taken to Secure the Repository
This repository is hosted on GitHub and access is limited to the course participants. Because the repository only contains coursework and does not include sensitive information, additional security measures such as advanced access controls are not necessary.
